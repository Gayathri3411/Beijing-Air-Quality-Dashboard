## 2. Description
This Power BI dashboard provides a comprehensive and interactive analysis of Beijing‚Äôs air quality by visualizing key pollutant levels, temperature trends, and humidity variations across multiple months. The report transforms complex environmental data into meaningful insights, enabling users to understand seasonal air quality patterns, pollutant fluctuations, and the weather conditions that impact pollution levels. Using intuitive visuals such as bar charts, KPI cards, line graphs, and interactive slicers, the dashboard highlights essential environmental indicators including Ozone, Indium Oxide, Sulphur, Benzene, CO(GT), Nitrous Oxide, and Nitrogen Dioxide. It also identifies key metrics such as the hottest day and most humid day captured in the dataset. ## 4. Data Source
The dataset contains environmental sensor readings collected across Beijing at multiple timestamps. It includes a wide range of pollutant and weather indicators such as:

### **Pollutant Metrics**
- Ozone  
- Indium Oxide  
- Sulphur  
- CO(GT)  
- Benzene  
- Nitrous Oxide  
- Nitrogen Dioxide  
- Non-Methane Hydrocarbons  

### **Weather Indicators**
- Temperature  
- Relative Humidity  
- Absolute Humidity  
- Date/Time Stamps  

The dataset was cleaned, transformed, and structured using Power Query to ensure accurate, reliable, and high-quality reporting in Power BI.

## 5. Features / Highlights

### üîπ Dashboard Highlights
- KPI cards displaying total pollutant values for:  
  Ozone, Indium Oxide, Sulphur, Absolute Humidity, Benzene, CO(GT)  
- Detection of **hottest day** and **most humid day** using DAX calculations  
- Seasonal Ozone analysis with **Ozone-by-Month Bar Charts**  
- CO(GT) and Nitrogen Dioxide **trend visualization using line charts**  
- Monthly average comparisons for:  
  Nitrous Oxide, Nitrogen Dioxide, Non-Methane Hydrocarbons  
- Clean, modern **dark theme** for improved readability  
- Fully interactive **slicers** including date range, month, and year filters
